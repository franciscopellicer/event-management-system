# 📊 Implementation of ADTs and a Custom Dataframe

This repository contains the specification and implementation of **Abstract Data Types (ADTs)** from scratch in Python, developed for the University of Murcia.

## 🎯 Technical Challenge
The main objective was to build complex data structures while managing memory at a low level, **with the strict restriction of not using Python’s native collections** (such as dictionaries, sets, or tuples).  
All the logic relies on the use of lists treated exclusively as contiguous *arrays*.

## 🛠️ Implemented Structures
* **Dataframe ADT:** Construction of a custom tabular structure for data storage and analysis.
* **Mail Manager ADT:** Implementation of storage and filtering logic.

## 💡 Demonstrated Skills
* Deep understanding of data structures and algorithmic efficiency.
* Explicit type declaration, visibility control, and responsibility delegation.

*Developed in Python 3.*
